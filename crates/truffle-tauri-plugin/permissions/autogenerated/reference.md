## Default Permission

Default permissions for the Truffle plugin

#### This default permission set includes the following:

- `allow-start`
- `allow-stop`
- `allow-is-running`
- `allow-device-id`
- `allow-devices`
- `allow-device-by-id`
- `allow-is-primary`
- `allow-primary-id`
- `allow-role`
- `allow-send-envelope`
- `allow-broadcast-envelope`
- `allow-handle-tailnet-peers`
- `allow-set-local-online`
- `allow-proxy-add`
- `allow-proxy-remove`
- `allow-proxy-list`
